# 按钮

## 自定义属性

### 类型

- 主要
- 次要
- 矩形

### 主题

- 属性名称：theme
- 枚举： blue/orange/grey

### 大小

- 大
- 中

## 用法

```pug
app-button(theme="blue") 确认支付
app-button(theme="orange") 确认支付
app-button(theme="purple") 确认支付
app-button(theme="red") 确认支付
app-button(theme="grey") 确认支付
app-button(theme="blue" size="large") 修改地址
app-button(theme="orange" size="large") 修改地址
app-button(theme="purple" size="large") 修改地址
app-button(theme="red" size="large") 修改地址
app-button(theme="grey" size="large") 修改地址
app-button(theme="blue" type="minor") 确认收货
app-button(theme="orange" type="minor") 确认收货
app-button(theme="purple" type="minor") 确认收货
app-button(theme="red" type="minor") 修改地址
app-button(theme="grey" type="minor") 修改地址
```
